# KeyFlip 🎹

Андроїд-додаток для конвертації тексту з кириличної розкладки клавіатури в латинську QWERTY.

---

## Про додаток

Буває що набираєш текст і забуваєш переключити мову — в результаті отримуєш `ghbdbn` замість `привіт`. KeyFlip миттєво виправляє такий текст — просто встав або введи його в додаток і отримай правильний результат.

Підтримуються дві розкладки:
- 🇺🇦 Українська → QWERTY
- 🇷🇺 Російська → QWERTY

---

## Можливості

- Конвертація тексту з української або російської розкладки в англійську QWERTY
- Авто-конвертація під час набору в реальному часі
- Копіювання результату в буфер обміну одним натисканням
- Підтримка великих і малих літер
- Простий і зрозумілий інтерфейс

## Встановлення

### Готовий APK

1. Завантаж файл `keyflip.apk` зі сторінки [Releases](../../releases)
2. Дозволь встановлення з невідомих джерел у налаштуваннях телефону
3. Відкрий завантажений файл і встанови

### Зібрати самостійно

**Вимоги:**
- Android Studio Hedgehog або новіше
- JDK 17+
- Android SDK (мінімум API 24)

**Кроки:**

```bash
git clone https://github.com/voipoddsns/KeyFlip.git
cd keyflip
```

Відкрий папку в Android Studio → `Build → Build APK`.

## Як це працює

Конвертер використовує таблицю відповідностей між кириличними символами і клавішами QWERTY-розкладки.

Наприклад для української:

| Кирилиця | QWERTY |
|----------|--------|
| й | q |
| ц | w |
| у | e |
| к | r |
| ... | ... |

Символи які не входять в таблицю (цифри, пробіли, розділові знаки) передаються без змін.
